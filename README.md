# dentalco

Le site est divisé en deux partie
-> le backend sert d'API entre le site et la base de données
-> le frontend qui est le site internet en dur

# backend
- npm i express
- npm i cors
- npm i bcrypt
- node index.js (lance l'api)


# frontend
- npm install -g @vue/cli
- npm install vue-router@4
- npm i vue3-carousel
- npm i axios
- npm run serve (lance le site)
